# Proyecto Tutor - Rama de Experimentales

Proyecto PHP + MySQLi para el cuadro de mando del tutor.

## Archivos principales
- config/conexion.php
- includes/funciones_tutor.php
- css/dashboard_tutor.css
- tutor.php
- lista_profesores.php

## Uso
http://localhost/proyecto_tutor_experimentales/tutor.php

## Integración con proyecto del profesor
Los botones **Más detalles** apuntan a:
`../proyecto_profesor_experimentales/resumen.php?id=ID`
